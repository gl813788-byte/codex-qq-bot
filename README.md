# GPT

这是一个初始化后的项目仓库。你可以在这里存放代码、文档、实验记录和项目配置。

## 项目结构

```text
.
├── README.md      # 项目说明
├── .gitignore     # 忽略本地临时文件和依赖产物
└── src/           # 源代码目录
```

## 开始使用

克隆仓库：

```bash
git clone https://github.com/gl813788-byte/GPT.git
cd GPT
```

后续可以把主要代码放在 `src/` 目录中，并根据项目语言继续添加依赖管理文件，例如：

- Python: `requirements.txt` 或 `pyproject.toml`
- Node.js: `package.json`
- Web 项目: `index.html`、`src/`、`public/`
